### Requirements

* Node >= 6.5

### Installation

1. Clone the git repo - `git clone git@github.com:jshmrtn/satelight.git`
2. Run `npm install`

### Development

Run `npm run start`

### Testing

Run `npm run test`

### Building

Run `npm run build:prod`
